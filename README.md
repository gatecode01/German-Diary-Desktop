# Mein Tagebuch (German Diary Desktop)

A simple but structured diary application built with Python.

## Overview 

Mein Tagebuch is a command-line diary desktop that allows users to store, view, and manage daily memories along with a mood rating system.
The project was created as a personal learning exercise to improve Python programming skills, especially working with file handling and user input.

## Features 

1- Create and save daily diary entries
2- View previous entries by date 
3- Delete today's entry 
4- Mood tracking system (Vibe selection)
5- Simple and user-friendly interface

## Technologies Used

1- Python 3
2- File handling (JSON)

## Purpose of the Project

This project was deveploped to practice:

1- Python fundamentals 
2- Working with files and data storage
3- Structuring a real small-scale application
4- Building a clean user interface logic
5- Preparing projects for GitHub portfolio
(Ausbildung IT preparation)
